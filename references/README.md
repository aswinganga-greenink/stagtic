# Stagtic Coding Competition - Offline References

Welcome to the offline repository for the Stagtic Coding Competition! Since internet access is disabled during the
event, we have provided the following official documentation archives, applications, and textbooks to help you solve the
algorithmic challenges.

## 1. Documentation & Textbooks by Language

### C & C++ (`c/` and `cpp/`)
* **Reference:** `cppreference-doc.tar.xz`
* The complete offline HTML dump of **cppreference.com**. Extract it and open `reference/en/index.html` in a web
browser.
* **Books:** `c_notes.pdf` & `cpp_notes.pdf`
* "Notes for Professionals" open-source textbooks covering everything from basics to advanced standard library
paradigms.

### Java (`java/`)
* **Reference:** `jdk-21-docs.zip`
* Official OpenJDK 21 API documentation. Extract the ZIP and open `api/index.html` in a web browser.
* **Book:** `java_notes.pdf`
* "Java Notes for Professionals" open-source textbook.

### Python (`python/`)
* **Reference:** `python-3.11-docs.zip`
* Official Python 3.11 documentation provided by the Python Software Foundation. Extract and open `index.html`.
* **Book:** `python_notes.pdf`
* "Python Notes for Professionals" open-source textbook.

## 2. Reading The Docs (Offline Tools)
Inside the `offline_browsers/` directory, you will find categorized tools for Linux, Windows, and macOS. These are
provided so you can comfortably browse the HTML documentation offline!

* **Web Browsers**: Official standalone Firefox offline installers (`Firefox_Setup.exe`, `firefox.tar.bz2`,
`Firefox.dmg`) for your OS so you can view any extracted `.html` documentation.
* **Offline Doc Browsers**: Also included are **Zeal** (for Windows/Linux) and **Dash** (for Mac). These are incredible,
native offline documentation browsers. If you install them *before* the internet is cut, you can download rich offline
"docsets" straight into the app!

## 3. Competitive Programming Handbook
* File: `CP_Handbook.pdf` by Antti Laaksonen
This is an incredible, industry-standard textbook that explains how to approach almost every algorithmic concept you
will face in competitive programming. It contains chapters on time complexity, sorting, greedy algorithms, dynamic
programming, graph theory, trees, mathematics, and advanced data structures.

Good luck, and may the best algorithm win!
